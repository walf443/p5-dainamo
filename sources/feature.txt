
特徴
====

dainamoは、perlで書かれたperfork型のworkerプロセスを管理するためのプログラムです。
dainamoは、これらの特徴をもちます。

graceful shutdown
------------------

SIGTERMを送信すると、jobがある程度終了するのを待ってからshutdownさせることができます。

logging
--------

起動/終了、jobを実行した内容、warnings、エラー終了等をログに記録させることができます。

柔軟な設定ファイル
------------------

perlがそのまま書けるので、柔軟に設定が書けます。

複数のプロセスの簡単な起動および終了
------------------------------------

一つの設定で、複数のworkerの起動の管理ができるので、プロジェクトのworkerスクリプト群の起動/終了が簡単になります。

admin portによる監視
--------------------

admin portを通してdainamoが管理している状態を知ることができるので、監視に使えます。
